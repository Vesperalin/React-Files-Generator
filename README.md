# React-files-generator

Automatically create folders and files in your React project.

## Features

Create file structure like this:

```bash
├── <given_name>
│   ├── index.ts
│   ├── <given_name>.styles.ts
│   ├── <given_name>.test.tsx
│   ├── <given_name>.tsx
│   └── <given_name>.types.ts
└──
```
